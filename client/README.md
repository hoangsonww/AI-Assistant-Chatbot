## Setup & Installation

### Development Setup

1. **Navigate to the client folder:**

   ```bash
   cd client
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the frontend development server:**

   ```bash
   npm start
   ```

   The application will start on [http://localhost:3000](http://localhost:3000).
